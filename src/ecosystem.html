GRANTS AND ECOSYSTEM

Apply for a grant

2020 saw the start of Swarm Grant Waves. Swarm grants and online events are supporting many interesting projects which are already building their products on top of Swarm. Swarm not only supports these projects financially, it also mentors the developers and founders. These projects also enjoy public attention, as Swarm tries to be a podium for them to shine on.

You can see the full list of current grantees here below. The last, third wave of grants was announced at the Swarm Live event. There is still time to apply if you haven't done so already.
